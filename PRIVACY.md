# Context Relay 隐私政策

**最后更新日期**: 2026年3月16日

## 1. 数据收集

Context Relay **不收集、不传输、不存储**任何用户数据到外部服务器。

### 本地处理
- 所有对话提取和处理均在用户浏览器本地完成
- 提取的内容仅通过浏览器剪贴板或页面注入方式使用
- 无任何数据发送到开发者的服务器

### Chrome Storage API
- 仅使用 `chrome.storage.local` 存储用户偏好设置（如提取轮次选择）
- 存储的数据完全保留在用户本地设备

## 2. 权限使用说明

| 权限 | 用途 |
|------|------|
| `activeTab` | 仅在用户点击扩展图标时访问当前标签页 |
| `scripting` | 用于在AI平台页面注入内容提取脚本 |
| `storage` | 存储用户本地设置偏好 |
| `clipboardWrite` | 将提取的对话内容写入剪贴板供用户粘贴 |

## 3. 第三方服务

本扩展不集成任何第三方分析、追踪或广告服务。

## 4. 数据安全

- 所有操作均在浏览器沙箱环境内完成
- 无远程服务器通信
- 无用户行为追踪

## 5. 用户权利

用户可以随时：
- 通过浏览器扩展管理器卸载本扩展
- 清除 `chrome.storage` 中存储的本地设置

## 6. 联系我们

如有隐私相关问题，请通过以下方式联系：
- GitHub Issues: https://github.com/manastone/context-relay/issues

## 7. 政策更新

本隐私政策可能随版本更新而调整，任何变更将在扩展更新时提供通知。
